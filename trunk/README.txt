README for ClusterViz

This software is released under LGPL license.

1. Introduction
ClusterViz is a Cytoscape plug-in for analysis and visualization of clusters from
graph. Based on another Cytoscape plug-in named MCODE, three different graph
clustering algorithms (IPCA, EAGLE and MCODE) were implemented in ClusterViz.
This plug-in is developed by Juan Cai and Gang Chen, and supported by Prof.
Jianxin Wang and Dr. Min Li, from Central South University.

2. Quick Start
Following is a short quick start for the usage of ClusterViz, detailed
information of the algorithms implemented in this plug-in can be found in
related papers.
	1. Download and install latest Cytoscape(www.cytoscape.org).
	2. Download Jar version of ClusterViz from our website.
	3. Copy or move clusterviz.jar file to the plugins directory in the
directory of Cytoscape.
	4. Start Cytoscape, import or open a network.
	5. Click plugin -> ClusterViz -> start.
	6. Click ClusterViz tab on left panel, choose IPCA and keep all parameters
as default.
	7. Click Analysis button.
	8. After analysis done, results will be shown in right panel.
